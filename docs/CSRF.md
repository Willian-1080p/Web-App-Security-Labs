# CSRF

## Objetivo

Compreender ataques Cross-Site Request Forgery.

## Descrição

Força um usuário autenticado a executar ações não desejadas.

## Exemplos

* Alteração de senha
* Transferência financeira
* Mudança de e-mail

## Mitigações

* CSRF Tokens
* SameSite Cookies
* Reautenticação
* Verificação Origin/Referer

## Ferramentas

* Burp Suite
* OWASP ZAP

## Referência OWASP

A01:2021 – Broken Access Control
