# Authentication Bypass

## Objetivo

Estudar falhas que permitem acesso indevido a aplicações.

## Vetores Comuns

- Credenciais padrão
- Falhas de lógica
- Session Fixation
- Session Hijacking
- Weak Password Policy

## Impactos

- Acesso não autorizado
- Escalada de privilégios
- Vazamento de dados

## Mitigações

- MFA
- Política forte de senhas
- Controle de sessão
- Rate Limiting

## Referência OWASP

A07:2021 – Identification and Authentication Failures
