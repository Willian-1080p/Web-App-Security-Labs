# Burp Suite

## Objetivo

Utilizar Burp Suite para análise de segurança em aplicações web.

## Ferramentas Principais

### Proxy

Interceptação de requisições.

### Repeater

Reenvio manual de requisições.

### Intruder

Automação de testes.

### Decoder

Codificação e decodificação.

### Comparer

Comparação de respostas.

## Casos de Uso

- SQL Injection
- XSS
- CSRF
- Auth Bypass
- Header Analysis
