# Web App Security Labs

Laboratório educacional focado em segurança de aplicações web, vulnerabilidades do OWASP Top 10 e técnicas de análise utilizadas por profissionais de Cybersecurity.

## Objetivo

Demonstrar vulnerabilidades comuns em aplicações web e apresentar métodos de identificação, exploração controlada e mitigação.

## Conteúdo

### SQL Injection

Identificação e mitigação de injeções SQL.

### Cross-Site Scripting (XSS)

Análise de XSS Refletido, Armazenado e DOM-Based.

### Cross-Site Request Forgery (CSRF)

Ataques de falsificação de requisição e suas proteções.

### Authentication Bypass

Falhas de autenticação e controle de acesso.

### OWASP Top 10

Principais riscos de aplicações web.

### Burp Suite Labs

Exemplos de uso do Burp Suite em análises de segurança.
